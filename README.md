# OpsOlive Landing Page

Static landing page for [OpsOlive](https://opsolive.com).

## Structure

```
ops-olive-landing-page/
├── index.html         # Main page
├── css/style.css      # Styles
├── assets/            # Images, icons, SVGs
└── CNAME              # Custom domain for GitHub Pages
```

## Development

Open `index.html` directly in a browser, or serve locally:

```bash
npx serve .
```

## Deployment

Deployed via GitHub Pages. Push to `main` to publish.
